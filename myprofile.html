Client                          : Johnson Controls Inc. (US)                        Sept’12 – Sept’13

Project Name             : Metasys Adaptor
 

Johnson Controls Inc. is a Building Efficiency Organization that delivers products, services and solutions that increase energy efficiency and lower operating costs in buildings for more than one million customers.

 

Primary function of MSB Adapter is to get Metasys data (via MSB) into Panoptix system. MSBAdapter will work as a mediator between the Panoptix applications and the existing Metasys Service Bus (MSB) (which will interact with actual Metasys hardware devices). The MSBAdapter will provide a well-known queue and blob repository for communication. The MSBAdapter will submit a client registration request to the MSB in order to define communication channels. Each channel will consist of a queue and a blob store. The MSBAdapter will receive requests to subscribe to point data, transform these into time series register requests and forward them on to the MSB. The MSBAdapter will monitor a queue for messages from the MSB. The MSB will send response messages for time series registration, time series updates and client register to the MSBAdapter via this queue. MSBAdapter will get the receive COV data for requested subscribed points from its MSB Response Queue and will insert write the same data in Azure Historian Database, so that Panoptix applications can use the same data and display the same as more meaningful information to the end user.
